# React Dashboard UI

This is a simple React dashboard app built with:

- React + Vite
- Tailwind CSS
- React Router
- Dummy data table with filtering

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 🌐 Deployment

This project can be deployed to [Vercel](https://vercel.com) easily.

## 📁 Folder Structure

```
src/
  components/
    Sidebar.jsx
    DataTable.jsx
  pages/
    Dashboard.jsx
  App.jsx
  main.jsx
  styles.css
```
